# MFOP Specifications — Peer Review

This directory contains the **Mahalaxmi Federation and Orchestration Protocol (MFOP) v1.0** specification, published for public peer review in 10 languages.

MFOP defines the wire protocol, data formats, cryptographic mechanisms, and behavioral requirements for federated AI orchestration across heterogeneous compute nodes.

## How to Provide Feedback

All feedback is welcome — technical corrections, translation accuracy, clarity improvements, or questions about the protocol design.

[**Open a feedback issue →**](https://github.com/mahalaxmi-ai/mahalaxmi/issues/new?template=spec-feedback.yml&labels=spec-feedback)

Please reference the section number and language in your issue.
